# DailyPlanner

Ежедневник для управления задачами, разработанный на Qt/C++.

## Описание

Приложение для планирования и управления ежедневными задачами с возможностью:
- Создания, редактирования и удаления задач
- Установки приоритетов и дат выполнения
- Фильтрации задач по статусу и приоритету
- Сортировки задач по дате и приоритету
- Сохранения и загрузки задач из файла

## Технологии

- C++23
- Qt (Core, Widgets)
- CMake
